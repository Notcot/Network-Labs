# Networking Labs

A collection of Packet Tracer labs and study notes for CCNA and networking in general.

## 📚 Repository Overview

This repository contains hands-on networking labs that I completed while studying for **CCNA**.

### Purpose
- Document my networking journey
- Practice and demonstrate core networking concepts
- Build a personal portfolio of practical skills

## 🛠️ Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI
- GNS3 (planned)

## 🎯 Learning Objectives
- Master IP addressing and subnetting
- Understand routing concepts (static & dynamic)
- Configure and troubleshoot networks
- Build strong foundational knowledge for cybersecurity

## 📌 How to Use This Repository
1. Navigate to the lab folder you're interested in
2. Read the `README.md` inside each lab for objectives and topology
3. View configuration files (`.txt`)
4. Open the `.pkt` file in Packet Tracer (if available)
